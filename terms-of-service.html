2. Description of Service

Clean My Prompt is a free, open-source web tool that:

  • Sanitizes sensitive data from text using client-side processing
  • Detects and replaces emails, API keys, passwords, IP addresses, credit cards, phone numbers, names, and other PII
  • Operates entirely in your browser with no server-side processing
  • Provides NLP-based smart detection and regex pattern matching
