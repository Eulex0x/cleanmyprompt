## ğŸ› Troubleshooting

**Issue:** "No such file or directory"
- **Fix:** Make sure you're in the project root directory where `wrangler.toml` exists

**Issue:** "Authentication required"
- **Fix:** Run `wrangler login` to authenticate

**Issue:** "Assets not loading"
- **Fix:** Ensure all files are in the root directory specified in `[assets]`

**Issue:** "404 on subpages"
- **Fix:** Cloudflare Workers automatically serves `.html` files. Access `/about` or `/about.html`
